# Tuần 6: Tài liệu hóa Frontend

## Review Code cuối cùng
Mọi thứ chạy ổn định. Nhánh `feature/frontend` đã được merge sạch sẽ vào `main`. Tôi dành thời gian đi rà soát lại toàn bộ console log trong quá trình debug và xóa hết chúng. Đảm bảo code sạch sẽ nhất khi thầy cô chấm.

## Viết báo cáo
Tôi tham gia cùng nhóm hoàn thiện file `Test_Cases.md` phần giao diện. Liệt kê rõ các thao tác người dùng. Tôi cũng hoàn thành bảng AI Audit Log của mình, trung thực ghi lại những lần mình "bí" React và phải nhờ ChatGPT viết hộ hàm xử lý mảng.

## Đóng gói
Thấy Vĩ vất vả gom file, tôi đã phụ Vĩ viết file `README.md` phần hướng dẫn chạy Frontend. Giải thích rõ lệnh `npm install` và `npm run dev`. Cảm giác nhìn sản phẩm hoàn thiện thật sự tuyệt vời.
