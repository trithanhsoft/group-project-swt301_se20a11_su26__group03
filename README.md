# Task Management System

This is the project repository for Group 03, created for the SWT301 Software Testing course.
The project consists of a Node.js/Express backend and a React (Vite) frontend.

## Features
- View all tasks
- Add new tasks
- Mark tasks as completed
- Delete tasks

## Setup Instructions

### 1. Backend Setup
```bash
cd backend
npm install
npm start
```
The API will run on `http://localhost:5000`.

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
The React application will be available at the local dev URL (usually `http://localhost:5173`).

## Testing

### Backend Tests
```bash
cd backend
npm test
```

### Frontend Tests
```bash
cd frontend
npm test
```
